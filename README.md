# Number Guessing Game

A simple, interactive Command Line Interface (CLI) Number Guessing Game built with Python. The computer randomly selects a number between 1 and 100, and the player must guess it within a limited number of attempts based on the chosen difficulty level. 

## 🎮 Features

* **Randomized Gameplay:** A new number is generated every time you play.
* **Difficulty Levels:** 
    * **Easy:** 10 chances to guess the number.
    * **Medium:** 5 chances to guess the number.
    * **Hard:** 3 chances to guess the number.
* **Smart Hints:** The game tells you whether your guess is too high or too low.
* **Error Handling:** Prevents the game from crashing if you accidentally type a letter instead of a number.
* **Replayability:** Easily start a new round immediately after winning or losing without having to restart the script.
